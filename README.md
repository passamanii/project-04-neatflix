# NetFlix Clone

    A Netflix-inspired web application built with vanilla JavaScript, HTML, and CSS.

## Features

- Profile Selection: Choose from 4 profiles with LocalStorage persistence
- Dynamic Catalog: Browse movies and TV series across multiple categories
- Video Previews: Hover over cards to see YouTube video previews
- Theme Toggle: Switch between light and dark modes
- Responsive Design: Works on all screen sizes

## Project Structure
- Tech Stack
- HTML5 / CSS3 / JavaScript (ES6+)
- YouTube Embed API
- LocalStorage for data persistence

## How to Use
- Select a profile from the home page
- Browse the catalog and hover over cards to preview videos
- Click to watch full videos on YouTube
- Toggle theme with the sun/moon button

## LocalStorage
    Stores active profile name, image, and theme preference.
